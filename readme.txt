Плагин для WordPress "Un-click counter - Счетчик скачиваний файлов".

== Description ==

Плагин для WordPress "Un-click counter - Счетчик скачиваний файлов" не требует никаких настроек и не использует базу данных.

По сути это счетчик кликов по ссылке, очень простой в установке и использовании. Для работы плагина достаточно присвоить ссылке на скачиваемый файл id="download" и в том месте, где вы хотите видеть счетчик количества скачиваний, разместить шорткод [un_click_counter]. Если у вашей ссылки другой ID, то шлрткод будет [un_click_counter id="id_ссылки"].

Пример:

<a href="https://kodabra.unchikov.ru/wp-content/uploads/2023/07/un-click-counter.zip" id="download">Скачать плагин</a> [un_click_counter]

<a href="https://kodabra.unchikov.ru/wp-content/uploads/2023/07/un-click-counter.zip" id="zx">Скачать плагин</a> [un_click_counter id="zx"]

Изменить стиль счетчика можно в файле стилей un-click-counter-style.css в каталоге плагина.


== Installation ==

1. Загрузите плагин "Un-click counter - Счетчик скачиваний файлов" на сайт WordPress и активируйте его. 

2. Пропишите в ссылке на скачиваемый файл id="download" (если у вашей ссылки отсутствует ID).

3. Разместите шорткод [un_click_counter] или [un_click_counter id="id_ссылки"] в том месте, где вы хотите видеть счетчик количества скачиваний файла.
